Want to switch themes?

What?

This purpose of this CodePen is to show how you can use javascript and CSS to effect HTML and, therefore change the look or layout of a page. I wrote this in javascript without the use of any javascript libraries (no jQuery).

How?

The basic concept is to create two (or more) sets of styles. One style is the default and uses dark texts on a light background. The second uses the opposite; light text on a dark background. I included two buttons that, when clicked, add or remove the additional class that will change the look of the page (theme).

document.querySelector
is used here to target the buttons we're after as well as the body. Once those are defined we use a combination of 
addEventListener
and 
classList
and then either 
add
or 
remove
to... add or remove the "dark" class from the class list of the body. Pretty simple.

Why?

I created this pen because my son is learning Javascript and wanted to understand some of the ways you could actually tie scripts to your page for something useful. This is one of many ways you can do that. Hopefully others can learn from this as well.
